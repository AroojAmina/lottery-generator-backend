# 🎟️ Lottery Ticket Generator

This project is a Python-based tool that generates marked lottery ticket PDFs from input text files.

## 🚀 Features
- Supports dynamic ticket formats (e.g., 58 balls / 60 balls)
- Reads structured input files
- Generates clean, printable PDF tickets
- Designed to be generic and reusable

## 🛠️ Tech Stack
- Python
- File Handling
- PDF Generation

## 🎯 Purpose
To automate the process of generating lottery tickets and reduce manual effort.

## 📌 Future Improvements
- GUI interface
- Web-based version (Flask/FastAPI)
- Database integration
