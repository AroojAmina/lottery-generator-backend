# Lottery Printer

This project is a simple lottery printer application.

## Project Structure
- `main.py`: Entry point for the application.
- `template.png`: Placeholder for the client template.
- `numbers.txt`: Placeholder for lottery numbers.
- `requirements.txt`: List of dependencies.
- `README.md`: Project documentation.

## How to Run
1. Ensure you have Python installed.
2. Run the application:
   ```
   python main.py
   ```
